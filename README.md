# Algoritmo de Huffman

### Instruções

##### Comprimir:
> huff -c [ARQUIVO]

##### Descomprimir:
> huff -d [ARQUIVO COMPRIMIDO]

##### Ajuda:
> huff -h

##### Estatísticas:
> huff -s [ARQUIVO COMPRIMIDO]

##### Redirecionar saída:
> huff -o [ARQUIVO DE SAIDA]
